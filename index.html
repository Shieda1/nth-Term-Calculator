What is the nth term?
The nth term is a formula that enables you to find any term in a sequence. The 'n' stands for the term number.
How to calculate nth term?
To calculate nth term you should use the following formula:
an = a1 + (n - 1) d
wehere:
an = nth term
a1 = 1st term
n = term number
d = common difference
Why use nth term?
The nth term is usually used in order to generate a sequence.
